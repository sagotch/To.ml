Module Toml.Table.Key

module Key: TomlInternal.Type.Key
type t = string
A strongly-typed key. Used for both section keys ([key1.key2]) and plain keys (key=...).

Keys don't quite follow the Toml standard. They may not contain the following characters: space, \t, \n, \r, ., [, ], double quote and #.

exception Bad_key of string
Exception thrown when an invalid character is found in a key.
val compare : 'a -> 'a -> int
Compare x y returns 0 if x is equal to y, a negative integer if x is less than y, and a positive integer if x is greater than y.
val of_string : string -> string
Builds a key from a plain string.
Raises TomlInternal.Type.Key.Bad_key if the key contains invalid characters.
val to_string : 'a -> 'a
Returns the key as a plain string.
