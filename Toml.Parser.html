Module Toml.Parser

module Parser: sig .. end

Parses raw data into Toml data structures
val parse : Lexing.lexbuf -> TomlInternal.Type.table
Given a lexer buffer, returns a Toml table.
Raises TomlParser.Error if the buffer is not valid Toml.
val from_string : string -> TomlInternal.Type.table
Given an UTF-8 string, returns a Toml table.
Raises TomlParser.Error if the string is not valid Toml.
val from_channel : Pervasives.in_channel -> TomlInternal.Type.table
Given an input channel, returns a Toml table.
Raises TomlParser.Error if the data in the channel is not valid Toml.
val from_filename : string -> TomlInternal.Type.table
Given a filename, returns a Toml table.
Raises
  • TomlParser.Error if the data in the file is not valid Toml.
  • Pervasives.Sys_error if the file could not be opened.
